（A级10.0\%，B级12.9\%，C级16.2\%，D级56.0\%），验证了模型的合理性与实用性。

\section{问题重述与分析}
% ============================================================

\subsection{问题背景}

中小微企业是国民经济的重要组成部分，但由于其规模相对较小、缺少抵押资产、财务信息不透明等原因，
银行在对其授信时面临较大的信息不对称和信用风险。银行通常依据信贷政策、企业交易票据信息以及
上下游企业的影响力，向实力强、供求关系稳定的企业提供贷款，并对信誉高、信贷风险小的企业给予
利率优惠。

某银行对确定放贷企业的贷款额度设定为10--100万元，年利率范围为4\%--15\%，贷款期限为1年。
附件1提供了123家有信贷记录企业的相关数据（含信誉评级和违约记录），附件2提供了302家无信贷
记录企业的相关数据，附件3提供了银行贷款年利率与客户流失率关系的2019年统计数据。

\subsection{问题分析}

\textbf{问题1：}对附件1中123家企业的信贷风险进行量化分析，并在年度信贷总额固定时给出信贷策略。
该问题的核心是\textbf{风险评估与策略优化}。由于附件1包含信誉评级和违约标签，
可利用AHP-熵权-TOPSIS模型构建综合评价体系，并以违约标签验证模型效果。

\textbf{问题2：}在问题1基础上，对附件2中302家无信贷记录企业进行风险评估，
并在年度信贷总额为1亿元时制定信贷策略。该问题的核心是\textbf{迁移评价}——
将问题1中建立的评价体系应用于无标签企业，在1亿元总额约束下按风险评分分配贷款。

\textbf{问题3：}考虑突发因素（如新冠疫情）对不同行业、不同类别企业的影响，
给出信贷调整策略。该问题的核心是\textbf{外部冲击量化}——通过行业分类
和冲击系数将突发事件影响纳入风险评估框架。

\subsection{特征预处理}

为提高模型稳定性，对原始特征进行了以下变换：
\begin{itemize}
    \item \textbf{对数变换：}对金额类指标（销项总金额、进项总金额、利润）和数量类指标
    （销项客户数、进项供应商数、交易网络广度）应用$\ln(1+x)$变换，降低极端值影响；
    \item \textbf{非线性惩罚：}对作废率指标应用$e^{3x}-1$指数变换，使高作废率（$>30\%$）
    受到指数级惩罚，更准确地反映发票异常行为的信用风险含义；
    \item \textbf{极端值截断：}对收入环比增长率限制在$[-1, 2]$范围内（即-100\%至+200\%），
    对利润率限制在$[-0.5, 0.5]$范围内，剔除微小企业因分母极小导致的异常比率；
    \item \textbf{Winsorize处理：}对所有指标进行1\%和99\%分位数截断。
\end{itemize}

\section{信贷风险评估模型}
% ============================================================

\subsection{AHP层次分析法确定主观权重}

AHP（Analytic Hierarchy Process）通过专家经验建立判断矩阵来确定指标的主观相对重要性。
本文建立了三层层次结构：目标层（信贷风险综合评分）$\rightarrow$准则层（企业实力B1、
企业稳定性B2、企业信誉B3、企业成长B4）$\rightarrow$指标层（15个具体指标）。

\subsubsection{准则层判断矩阵}

根据信贷风险评价的领域知识，企业实力最为重要，其次为企业信誉、企业稳定性，最后为企业成长。
准则层判断矩阵如下：

\begin{equation}
\mathbf{B} = \begin{bmatrix}
1 & 3 & 2 & 4 \\
1/3 & 1 & 1/2 & 2 \\
1/2 & 2 & 1 & 3 \\
1/4 & 1/2 & 1/3 & 1
\end{bmatrix}
\end{equation}

采用几何平均法计算权重，结果为：
$\mathbf{w}_B = (0.467, 0.160, 0.278, 0.095)^T$。
一致性检验结果为：$\lambda_{\max}=4.031$，$CI=0.011$，$CR=0.0093 < 0.10$，通过一致性检验。

\subsubsection{综合权重合成}

各准则下指标采用均权分配，最终AHP综合权重通过准则权重与准则内指标权重相乘得到。

\subsection{熵权法确定客观权重}

熵权法基于信息论原理，通过指标数据的离散程度来确定客观权重。差异系数越大，
说明该指标提供的信息量越大，应赋予更大权重。

具体步骤如下：
\begin{enumerate}
    \item 计算第$j$个指标下第$i$个企业的比重：$p_{ij} = x'_{ij} / \sum_{i=1}^{n} x'_{ij}$
    \item 计算第$j$个指标的信息熵：$e_j = -\frac{1}{\ln n}\sum_{i=1}^{n} p_{ij}\ln(p_{ij} + \varepsilon)$，
          其中$\varepsilon=10^{-12}$防止$\ln(0)$
    \item 计算差异系数：$d_j = 1 - e_j$
    \item 计算权重：$w_j^{\text{Ent}} = d_j / \sum_{j=1}^{m} d_j$
\end{enumerate}

\subsection{组合赋权}

为兼顾主观经验与客观数据，采用线性加权法进行组合赋权：

\begin{equation}
w_j = \alpha \cdot w_j^{\text{AHP}} + (1-\alpha) \cdot w_j^{\text{Ent}}, \quad \alpha = 0.6
\end{equation}

其中$\alpha=0.6$的选择是基于以下考量：在信贷风险评价中，专家的经验判断（AHP）
通常比纯数据驱动的权重（熵权法）更可靠，尤其是在某些指标（如收入环比增长率）
可能因数据噪声而产生过高熵权时。AHP占比较高可以有效地抑制噪声指标对模型的
过度影响。

\subsection{TOPSIS综合评价}

TOPSIS（Technique for Order Preference by Similarity to Ideal Solution）通过计算
各评价对象与理想解和负理想解的距离来进行排序。

算法步骤如下：

\textbf{步骤1：指标正向化。}将各类指标统一转换为极大型（越大越好）。四种指标类型的
转换公式如下：

\begin{itemize}
    \item 极大型：$x'_{ij} = \dfrac{x_{ij} - \min(x_j)}{\max(x_j) - \min(x_j)}$
    \item 极小型：$x'_{ij} = \dfrac{\max(x_j) - x_{ij}}{\max(x_j) - \min(x_j)}$
    \item 中间型：$x'_{ij} = 1 - \dfrac{|x_{ij} - x_{\text{best}}|}{\max|x_{ij} - x_{\text{best}}|}$
    \item 区间型：在区间$[a,b]$内取1，区间外线性递减
\end{itemize}

\textbf{步骤2：向量归一化。}
\begin{equation}
z_{ij} = \frac{x'_{ij}}{\sqrt{\sum_{i=1}^{n} (x'_{ij})^2}}
\end{equation}

\textbf{步骤3：加权矩阵。}
\begin{equation}
v_{ij} = w_j \cdot z_{ij}
\end{equation}

\textbf{步骤4：确定正、负理想解。}
\begin{equation}
V^+ = (\max_i v_{i1}, \max_i v_{i2}, \ldots, \max_i v_{im}), \quad
V^- = (\min_i v_{i1}, \min_i v_{i2}, \ldots, \min_i v_{im})
\end{equation}

\textbf{步骤5：计算欧氏距离。}
\begin{equation}
D_i^+ = \sqrt{\sum_{j=1}^{m}(v_{ij} - v_j^+)^2}, \quad
D_i^- = \sqrt{\sum_{j=1}^{m}(v_{ij} - v_j^-)^2}
\end{equation}

\textbf{步骤6：计算相对贴近度（风险得分）。}
\begin{equation}
C_i = \frac{D_i^-}{D_i^+ + D_i^-}, \quad C_i \in [0, 1]
\end{equation}

$C_i$越大，表示企业越接近正理想解，即信用状况越好、风险越低。
按$C_i$降序排列即为企业信用风险从低到高的排名。

是合理的：模型基于客观发票数据，而银行评级基于信贷员的主观综合判断，两者维度互补。

\section{信贷策略制定模型}
% ============================================================

\subsection{利率-流失率关系建模}

利用附件3中29组利率-流失率数据，对A、B、C三个信誉等级分别进行二次多项式拟合：

\begin{equation}
L_k(r) = a_k r^2 + b_k r + c_k, \quad k \in \{A, B, C\}
\end{equation}

拟合结果：
\begin{itemize}
    \item A级：$L_A(r) = -76.41r^2 + 21.98r - 0.697$，$R^2 = 0.993$
    \item B级：$L_B(r) = -67.93r^2 + 20.21r - 0.650$，$R^2 = 0.995$
    \item C级：$L_C(r) = -63.94r^2 + 19.57r - 0.639$，$R^2 = 0.995$
\end{itemize}

三个等级的$R^2$均大于0.99，拟合效果优异。流失率随利率递增：在4\%利率时流失率接近0\%，
在15\%利率时流失率约为88\%--92\%。

\subsection{信贷额度与利率分配策略}

\textbf{额度分配原则：}
\begin{itemize}
    \item 风险等级为D的企业不予放贷（贷款额度为0）；
    \item A、B、C级企业的贷款额度按TOPSIS得分比例分配，限制在10--100万元；
    \item 当实际总额与预算有偏差时，通过迭代再分配算法调整至收敛（误差$<0.05\%$）。
\end{itemize}

\textbf{利率定价原则：}
\begin{itemize}
    \item 基准利率：A级4\%，B级6.5\%，C级10\%；
    \item 在基准利率基础上，根据TOPSIS得分进行微小浮动（$\pm 1\%$），得分高的企业享受更低利率；
    \item 利率严格限定在4\%--15\%范围内。
\end{itemize}

\section{突发因素影响下的信贷调整策略}
% ============================================================

\subsection{行业分类方法}

从企业名称中提取行业关键词，将302家企业分为四类：
\begin{itemize}
    \item \textbf{严重负面行业（3家）：}旅游、酒店、餐饮、影院等——直接受到疫情封控冲击，客流量骤降；
    \item \textbf{中度负面行业（69家）：}建筑、制造、批发零售、物流运输等——受供应链中断和消费萎缩影响；
    \item \textbf{正面/轻影响行业（43家）：}医疗、科技、信息技术、电子商务等——需求增长或影响有限；
    \item \textbf{其他行业（187家）：}无法从名称中识别行业类型，默认为中性影响。
\end{itemize}

\subsection{冲击系数设定}

基于文献研究\cite{ref_covid}，新冠疫情对不同行业的影响差异显著：
\begin{itemize}
    \item 严重负面行业：冲击系数 $S=1.30$（风险上调30\%）
    \item 中度负面行业：冲击系数 $S=1.15$（风险上调15\%）
    \item 其他行业：冲击系数 $S=1.00$（维持不变）
    \item 正面/轻影响行业：冲击系数 $S=0.90$（风险下调10\%）
\end{itemize}

\subsection{调整算法}

\begin{enumerate}
    \item 调整后风险得分：$C'_i = C_i / S_k$，其中$S_k$为企业$i$所属行业$k$的冲击系数；
    \item 按调整后得分重新排序并重新划分风险等级；
    \item 利率调整：原始等级基准利率 + 行业利率调整项（严重负面+2\%，中度负面+1\%，正面-1\%）；
    \item 额度按调整后得分比例重新分配（10--100万元约束），迭代至预算收敛。
\end{enumerate}

\section{灵敏度分析}
% ============================================================

\subsection{权重扰动分析}

为检验模型对权重变化的鲁棒性，对每个指标的权重进行100次随机$\pm 10\%$扰动，
计算扰动后排名与原始排名的Spearman和Kendall秩相关系数：

\begin{itemize}
    \item Spearman相关系数：均值0.965，最小值0.949
    \item Kendall相关系数：均值0.848，最小值0.815
\end{itemize}

Spearman相关系数均值$>0.85$，表明模型排名对权重扰动具有高度稳健性。

\subsection{指标移除分析}

逐一移除每个指标后重新计算TOPSIS排名，观察排名变化。由于TOPSIS方法的几何特性，
移除单个指标会改变整个评价空间的几何结构，但各指标移除后的排名相关性分布反映了
各指标的相对影响程度。结果再次验证了组合赋权（$\alpha=0.6$）策略能有效减少
任何单一指标对最终排名的过度影响。

\section{模型的优点与不足}
% ============================================================

\subsection{模型优点}

\begin{enumerate}
    \item \textbf{主客观结合：}AHP与熵权法的组合赋权兼顾了专家经验与客观数据，避免了
    纯主观赋权的主观性和纯客观赋权可能被噪声数据误导的风险。
    \item \textbf{可解释性强：}TOPSIS评价结果直观易懂（0--1得分），风险等级与违约率
    呈严格单调关系，便于银行信贷人员理解和应用。
    \item \textbf{计算高效：}利用SQLite数据库和SQL聚合查询处理百万级发票数据，
    特征工程在数秒内完成，模型训练和评分可实时进行。
    \item \textbf{鲁棒性好：}灵敏度分析验证了模型对权重扰动的稳健性，排名结果可信度高。
    \item \textbf{可扩展性：}本文的行业分类+冲击系数框架可推广到其他突发事件
    （如自然灾害、贸易摩擦等）的风险评估场景。
\end{enumerate}

\subsection{模型不足与改进方向}

\begin{enumerate}
    \item \textbf{行业分类精度有限：}由于企业名称经过脱敏处理（如``***电器销售有限公司''），
    62\%的企业无法精确归类，影响了问题3调整策略的精度。可考虑利用发票中的交易对手信息
    辅助行业识别。
    \item \textbf{时序信息利用不足：}当前模型主要使用聚合统计数据，未充分利用发票的时间序列
    特征（如季节性、趋势变化）。引入LSTM或Transformer等时序模型可能提升违约预测精度。
    \item \textbf{收入环比增长率熵权偏高：}该指标因方差过大导致熵权较高（0.388原始值），
    虽然通过AHP调节（$\alpha=0.6$）后降至0.184，但仍有一定影响。建议改用更稳健
    的成长性指标（如多期移动平均增长率）。
    \item \textbf{未利用上下游网络信息：}发票数据中隐含了企业间的交易网络结构，
    图神经网络（GNN）等方法可利用网络拓扑信息提升风险传染的识别能力。
\end{enumerate}

\section{结论}
% ============================================================

本文针对中小微企业信贷决策问题，构建了基于AHP-熵权组合赋权的TOPSIS综合评价模型，
完成了以下工作：

\begin{enumerate}
    \item 从进销项发票数据中提取了15个企业级财务与行为指标，构建了涵盖企业实力、
    稳定性、信誉和成长四个维度的风险评价指标体系；
    \item 建立了AHP-熵权组合赋权（$\alpha=0.6$）的TOPSIS风险评估模型，
    模型验证显示风险等级与违约率严格单调递增，D级违约率高达56.0\%；
    \item 基于附件3数据拟合了利率-流失率二次函数关系（$R^2 > 0.99$），
    并制定了差异化信贷策略——在总额约束下按风险得分分配额度和梯级利率；
    \item 引入行业冲击系数框架，量化了新冠疫情对302家企业信贷策略的差异化影响，
    严重负面行业额度平均下降44\%，正面行业额度上升22\%；
    \item 灵敏度分析验证了模型的鲁棒性（Spearman $\rho = 0.965$）。
\end{enumerate}

模型结果为银行在控制信贷风险的前提下最大化收益提供了量化决策支持，
为中小微企业融资难问题提供了一种基于数据驱动的解决方案。
